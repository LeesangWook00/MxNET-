### **2. 🤖 MxNET/PyTorch 활용 선형회귀 객체지향 분석**
> [cite_start]**선형회귀 모델의 밑바닥(Scratch) 구현과 엔진(Concise) 활용 비교 분석** > *(PBL 팀 프로젝트)* [cite: 1]

* [cite_start]**📅 개발기간** : 2025.06 ~ 2025.06 [cite: 40]
* [cite_start]**👑 핵심 역할** : 공동 연구 및 보고서 작성, 선형회귀 알고리즘 구현 및 성능 비교 분석 [cite: 1, 31, 37]
* [cite_start]**💻 개발 환경** : Python, PyTorch, MxNET Framework [cite: 1, 35, 36]

#### **⚙️ 주요 분석 내용**
* [cite_start]**객체지향 디자인 패턴 적용:** `Module`, `DataModule`, `Trainer` 등 독립적인 클래스 설계를 통한 기계학습 파이프라인 구조 분석 [cite: 4, 6, 8]
* **알고리즘 구현 비교:**
    * [cite_start]**Scratch:** 파라미터 초기화부터 역전파(Backward), 경사하강법(SGD)까지 직접 구현하여 동작 원리 파악 [cite: 31, 32]
    * [cite_start]**Concise:** 프레임워크 내장 API를 활용한 효율적인 모델 생성 및 최적화 방법 습득 [cite: 31, 34]
* [cite_start]**최적화 알고리즘 학습:** 미니배치 스토캐스틱 경사하강법(SGD)의 수식적 이해 및 학습률(lr) 튜닝 실습 [cite: 2, 3]

#### **✨ 기술적 성장**
* [cite_start]단순 모델 활용을 넘어, 클래스 간의 입력·동작·출력 관계를 분석하며 **확장성 있는 코드 설계 능력**을 배양함 [cite: 12, 19, 22]
* [cite_start]복잡한 인공지능 모델(영상, 언어 등)을 다루기 위한 **기계학습의 기초 파이프라인(Data-Model-Loss-Optimization) 전 과정을 완벽히 이해**함 [cite: 1, 40]
